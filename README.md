## 📌 Course Overview

This repository contains all lab submissions for **CPE343 - Computer Organization and Architecture**. The labs cover:

- **MIPS Assembly Programming** (using MARS simulator)
- **Verilog HDL** for combinational and sequential circuits
- **Single-Cycle MIPS Datapath** implementation

---

## 🛠️ Topics Covered

| Lab # | Title |
|-------|-------|
| Lab 01 | Introduction to MIPS Assembly |
| Lab 02 | Strings and Conditional Execution |
| Lab 03 | Integer Arithmetic |
| Lab 04 | Flow Control |
| Lab 05 | Introduction to Verilog HDL - Combinational Circuits |
| Lab 06 | Combinational and Sequential Circuits |
| Lab 07 | Instruction Fetch Module of MIPS Datapath |
| Lab 08 | Single Cycle MIPS Datapath (R-type & Load/Store) |
| Lab 09 | Single Cycle MIPS Datapath |
| Lab 10 | Single Cycle MIPS Datapath (ALU & Control Unit) |
| Lab 11 | Single Cycle MIPS Datapath (Data Memory & Branch) |
| Lab 12 | 32-bit 4-1 Multiplexer in Verilog |
